delta function

step function

$\delta$-函数常用于表示质点、点电荷等理想模型的密度函数[电荷密度]，该函数在除了零以外的点取值都等于零，而其在整个定义域上的积分等于1。$\delta$ function不规定在0的取值，是因为这个值无法严谨的表示出来，不能笼统的定义为正无穷。严格来说$\delta$函数不能算是一个函数，因为满足以上条件的函数是不存在的。
