How Naive Bayes Classifier Works[1]

How Naive Bayes Classifier Works[2]

The Naive Bayesian classifier is based on Bayes’ theorem with the independence assumptions between predictors (data attributes). A Naive Bayesian model is easy to build, with no complicated iterative parameter estimation which makes it particularly useful for very large datasets. Despite its simplicity, the Naive Bayesian classifier often does surprisingly well and is widely used because it often outperforms more sophisticated classification methods.

The meaning of “Naive”:

  1. Features are independent to each other.
  2. Features are equally important.

Algorithm

The classifer uses bayes rule to make predictions. Bayes classifier assume that the effect of the value of a predictor (attribute x) on a given class (c) is independent of the values of other predictors.

And we have a unknown case as following, and we want to know whether it is ok to play golf?

outlook temperature humidity windy play golf
rainy mild normal true ?

  1. For first class play golf = Yes:

    Among the total 14 records, the number of play-golf-yes is 9, so $P(Yes)=\frac{9}{14}$

    • For first arrtribute outlook:

      Among 9 yes records, there are 2 records where outlook is rainy, so

      $P(outlook=rainy|Yes)=\frac{2}{9}$

    • For second attribute temperature:

      Among 9 yes records, there are 4 records where temperature is mild, so $P(mild|Yes)=\frac{4}{9}$

    • For second attribute Humidity:

      Among 9 yes records, there are 6 records where humidity is normal, so

      $P(normal|Yes)=\frac{6}{9}$

    • For second attribute windy:

      Among 9 yes records, there are 3 records where windy is true, so

      $P(windy|Yes)=\frac{3}{9}$

    So $P(Yes|X)=P(rainy|Yes)P(mild|Yes)P(normal|Yes)P(windy|Yes)P(Yes)=\frac{2}{9}\frac{4}{9}\frac{6}{9}\frac{3}{9}\frac{9}{14}=0.0141099347$

  2. For second class play golf is No:

    $P(No|X)=P(rainy|No)P(mild|No)P(normal|No)P(windy|No)P(No)=0.010285714$

Normalize:

$P(Yes)=\frac{0.0141099347}{0.0141099347+0.010285714}=0.578368999$

$P(No)=\frac{0.010285714}{0.0141099347+0.010285714}=0.421631001$

So we can predict it is ok to play golf when outlook is rainy, temperature is mild, the humidity is normal and it is windy.
